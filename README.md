# navora Bau & Sanierung – Website

Einfache, statische One-Page-Website für das Innenausbau- und Sanierungsunternehmen **navora Bau & Sanierung**.

## Projekt starten

1. Öffne diese Mappe in VS Code (bereits erfolgt).
2. Öffne die Datei `index.html`.
3. Nutze in VS Code den Befehl **"Open with Live Server"** (oder einen ähnlichen lokalen Webserver), oder öffne die Datei direkt im Browser per Doppelklick.

## Dateien

- `index.html` – Struktur der Seite (Hero, Leistungen, Projekte, Über uns, Kontakt)
- `styles.css` – Design und Layout (responsive, dunkler, moderner Stil)
- `script.js` – Kleine Interaktionen (Navigation, Jahr im Footer, Formular-Hinweis)

## Dinge, die du anpassen solltest

- Firmendaten (Adresse, Telefon, E-Mail) im Kontakt-Bereich anpassen
- Platzhalter-Texte ggf. an dein Angebot (Leistungen, Projekte) anpassen
- Platzhalter-Bereiche für Projektfotos mit echten Bildern ersetzen
